# Old Fashioned

[![Build Status](https://travis-ci.org/myles/old-fashioned.svg?branch=master)](https://travis-ci.org/myles/old-fashioned)

## Development

	$ bundle install --path vendor/bundle
	$ npm install
	$ bower install

## Testing

	$ grunt test

## Build

    $ grunt build

## Build the Web Site

	$ grunt site
